REPLICUBE — MODELO DE EJECUCIÓN

El código se ejecuta una vez por cada celda (-6 a 6 en x, y, z).

Las variables x, y, z están disponibles directamente.

return N coloca un voxel de color N. Sin return o return 0/negativo = vacío.

-- Cubo rojo macizo return 7 -- Mitad superior verde, inferior marrón if y > 0 then return 11 else return 15 end
PALETA (1–16, wrap automático)

0 o negativo = vacío. Valores >16 hacen wrap: 17→1, 18→2…

sign(a) → 1, −1 ó 0

EJEMPLOS

Rugby ball (ejemplo oficial):

if abs(z) < 3 and abs(y) < 3 then if abs(z) < 2 and x % 2 == 0 and abs(x) < 3 and y == 2 then return 1 end return 15 end

Escaleras:

if x == y and z > 0 then return 2 end if x == -y and z < 0 then return 2 end if z == 0 then return 7 end return 0

Esfera hueca:

local d = sqrt(x*x + y*y + z*z) if inrange(d, 4, 6) then return 11 end

Esponja de Menger:

local nx = abs(x) % 3 local ny = abs(y) % 3 local nz = abs(z) % 3 local h = btoi(nx==1 and ny==1) + btoi(ny==1 and nz==1) + btoi(nx==1 and nz==1) if h == 0 then return 6 end

Tablero en el suelo:

if y == -6 then if (x + z) % 2 == 0 then return 1 else return 3 end end
REPLIPAINT — MODELO DE EJECUCIÓN

El código se ejecuta una vez por cada pixel del área visible.

Las variables x, y están disponibles directamente.

return N pinta el pixel. 0 o sin return = transparente.

-- Tablero de ajedrez if (x + y) % 2 == 0 then return 1 else return 3 end
FUNCIÓN EXTRA

fill(colorIndex) — rellena el fondo con un color de paleta

PALETA (1–16)
EJEMPLOS

Círculos concéntricos:

local d = sqrt(x*x + y*y) return (floor(d / 6) % 16) + 1

Onda de seno:

fill(3) local wave = floor(sin(x * 0.15) * 15) if abs(y - wave) <= 1 then return 8 end

Degradado diagonal:

return ((x + y) % 16 + 16) % 16 + 1
